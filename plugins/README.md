# Plugins Directory

This directory is reserved for Ansible plugins that extend the collection's functionality.

## Plugin Types

- `modules/` - Custom Ansible modules
- `module_utils/` - Shared utility code for modules
- `inventory/` - Dynamic inventory plugins
- `filter/` - Jinja2 filter plugins
- `lookup/` - Lookup plugins

Currently, this collection does not include custom plugins but the structure is in place for future extensions.
